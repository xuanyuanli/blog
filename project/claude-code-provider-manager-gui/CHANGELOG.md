## 版本说明

### 版本号规则
- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 发布周期
- **主版本**: 每 6-12 个月
- **次版本**: 每 1-3 个月
- **修订版本**: 根据需要发布

### 兼容性承诺
- 同一主版本内的次版本和修订版本保持向下兼容
- 主版本升级可能包含破坏性变更
- 重大变更会提前通知并提供迁移指南
